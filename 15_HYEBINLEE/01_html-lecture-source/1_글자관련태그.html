문단을 나누는 태그

문단 영역을 나누는 태그로는 p태그와 pre태그가 있다. p태그는 문단영역을 나누는 태그이지만 한 개의 공백만 표시하며 줄 바꿈 입력을 별도의 태그로 지정해주며 pre태그는 여러 칸 띄우기 혹은 줄바꿈 등을 포함하여 내용 그대로 표현하는 태그이다..

function foo() { close.log('foo'); } 
        
            function foo() {
                close.log('foo');
            }
